# audio — 경기 효과음 (MVP: 휘슬·스파이크·서브)

`assets/audio/serve.wav`·`spike.wav`·`whistle.wav` 는 현재 **무음 플레이스홀더**다(배선·빌드 검증용).
**같은 파일명으로 실제 음원을 덮어쓰면 코드 수정 없이 바로 소리가 난다.**

| 파일 | 언제 | 비고 |
|---|---|---|
| `serve.wav` | 서버가 공을 때리는 순간(서브 임팩트) | 에이스·서브범실 포함(서브 컨택은 같음) |
| `spike.wav` | 스파이크 **강타** 컨택 | **페인트(연타)·소프트샷은 안 남**(`!to.soft`) — 사용자 요청 |
| `whistle.wav` | 랠리가 끝나 점수가 날 때(종결 휘슬) | 랠리당 1회 |

## 음원 교체 가이드
- 권장: 짧게(0.1~0.4s), mono, 44.1kHz. 포맷은 `wav`/`mp3`/`m4a` 모두 가능(expo-audio).
- **CC0/로열티프리** 권장(freesound.org 등) — 라이선스 확인 필수.
- `.mp3` 등 다른 확장자로 넣으면 `audio/sfx.ts` 의 `SOURCES` require 경로만 바꾼다.
- 음량은 `audio/sfx.ts` `initSfx()` 의 `p.volume` 에서 조정.

## 동작
- 엔진 무의존(UI 레이어). 보드(`components/MatchCourt.tsx`)가 구간 이벤트에 맞춰 `playSfx()` 호출.
- 설정 화면 **"효과음"** 토글로 끔/켬(기본 켜짐, 스토어 `sfxEnabled` 영속).
- **폰 무음 스위치 존중** — 무음 모드면 토글이 켜져 있어도 소리 안 남(관전형 — 조용히 보고 싶은 사람 배려).

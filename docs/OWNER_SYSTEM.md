## 0. 왜 넣는가 (서사적 목표)

- 지금 "우승 야망 선수가 하위권 팀을 떠난다"는 FA 수식 안에서 **조용히** 일어난다.
  플레이어는 시즌말 명단에서 빠진 이름을 보고서야 안다. 드라마가 수식 속에 숨어 있는 것.
- 면담은 그 수식을 **장면**으로 꺼낸다: "구단주님, 저는 우승이 하고 싶습니다." — 여기서
  무엇을 약속할지, 약속이 통할지, 실패하면 마음이 더 떠날지. 의사결정의 무게(기둥 4).
- 감독 건의·팬심은 그 결정의 **파장**이다. 선수 하나를 벤치에 앉히는 결정이
  감독과의 신경전 → 팬 분노 기사 → 관중 감소 → 예산 타격 → 선수 이탈로 번진다.
  관전형 게임에서 "보는 맛"은 경기만이 아니라 이런 연쇄에서 나온다.

## 1. 선수 면담 (Interview)

### 1.1 불만은 이미 존재한다 — 새로 만들지 않는다
선수마다 FA 성향(`faPref`: money/winnow/loyal/minutes/hometown)이 이미 있다.
면담 시스템은 성향과 현실의 **불일치**를 불만으로 표면화한다:

| 성향 | 불만 조건 (파생, 저장 없음) | 면담에서 하는 말 |
|---|---|---|
| winnow(우승) | 팀 최근 2시즌 평균 순위 5위 이하 | "우승권 팀에 가고 싶습니다" |
| minutes(출전) | 최근 10경기 출전 3경기 이하(벤치) | "뛰고 싶습니다" |
| money(연봉) | 연봉 < 시장가치 × 0.75 | "제 가치를 인정받고 싶습니다" |
| hometown(연고) | 선호팀이 따로 있음 | "고향 팀에서 뛰는 게 꿈이었습니다" |
| loyal(충성) | 거의 불만 없음 | (면담 요청 시 "괜찮습니다, 구단주님") |

스쿼드/선수 상세에 **불만 뱃지**(😟)를 표시 — 면담할 이유를 플레이어가 발견하게.

### 1.2 면담 빈도 — 쿨다운 + 감정 (2026-06-17 쿨다운 추가)
- **쿨다운**: 한 선수와 면담하면(성공·거절·결렬 무관) `TALK_COOLDOWN_DAYS=24`일 동안 재면담 불가
  — 하루에도 여러 번 들볶는 것 방지(사용자 요청). UI: 면담 버튼 비활성화 + 남은 일수 표시.
  store `talkCooldown`(playerId→가능일), 시즌말 `currentDay` 0 리셋과 함께 초기화.
- 쿨다운이 풀려도 **선수가 거절할 수 있다**: 거절 확률 = 기저 10% + 같은 시즌 면담마다 +25%
  + 직전 면담 실패 +30%. → 들볶은 이력이 쌓이면 "드릴 말씀 없습니다". 쿨다운(기계)+감정(사람) 이중.
- 시드: `talk:{playerId}:{season}:{n}` (n = 그 시즌 면담 회차). 결정론.
- **면담 UI(2026-06-17)**: 시스템 Alert → 앱 테마 커스텀 모달(불만 뱃지·대사·약속 카드 선택지·결과 화면).

### 1.3 설득 — 약속 카드와 성향 매칭
면담이 열리면 불만 주제가 나오고, 구단주는 **약속 카드** 중 하나를 고른다:

| 카드 | 통하는 성향 | 비고 |
|---|---|---|
| "전력을 보강하겠다" | winnow | 다음 시즌 FA 영입 발생 시 신뢰 +(이행), 없으면 다음 면담 거절↑(공약 파기) |
| "주전을 보장하겠다" | minutes | 벤치 지시와 모순되면 즉시 역효과 |
| "재계약 때 성의를 보이겠다" | money | 재계약 제시액이 시장가 이상이면 이행 |
| "당신은 이 구단의 심장이다" | loyal·근속 4+ | 프랜차이즈 정서 자극 |

성공 확률(placeholder) = `0.35 + 0.4×(카드-성향 매칭) + 0.1×구단성적보정 − 0.15×기존실패횟수`, clamp 0.1~0.9.

### 1.4 결과 — 성공도 실패도 FA 수식에 꽂힌다
- **성공**: 그 선수의 FA 잔류 항(offerScore의 loyalty/해당 동기 항)에 보정 `+P` (시즌 한정).
  불만 뱃지 해소(😊). 뉴스: "○○, 구단과 면담 후 '동기부여 됐다'".
- **실패(역효과)**: 잔류 보정 `−P×0.7`. 구단의 진심을 확인하고 실망 — 이탈 의향이 **오히려 오른다.**
  실패 누적 2회면 사실상 마음이 떠난 상태(뱃지 💔). 뉴스: "○○ 면담 결렬… 거취 주목".
- 저장: `interviews: { playerId, season, day, topic, card, ok }[]` — 시즌말 FA 판정이 읽고, 시즌 경계에 정리(이행 판정 후).

## 2. 감독 건의 — "이 선수, 주전에서 빼주시죠"

### 2.1 왜 감독을 거치는가
구단주가 라인업을 직접 짜면 감독 시스템이 장식이 된다. **건의**로 만들면
감독의 성향·자존심이 게임 요소가 되고, 거절당하는 경험이 감독 교체 동기가 된다.

### 2.2 판정 — 감독은 합리와 소신 사이에서 답한다
수락 확률(placeholder):
```
base 0.5
+ 0.3 × 대체자 합리성     // 벤치 후보 vs 대체 선수 OVR 격차가 작을수록 수긍
− 0.4 × 에이스 보호       // 팀 내 OVR 1~2위를 빼라면 강하게 저항
− 0.2 × (charisma−50)/50  // 카리스마 높은 감독 = 소신 — 외풍에 흔들리지 않음
+ 0.2 × 사유 타당성       // "재계약 거부 예정" 같은 명분
```
시드: `bench:{playerId}:{season}:{day}`.
- **건의 쿨다운(2026-06-17)**: 주전/벤치 건의는 한 번 하면(수락·거절 무관) `BENCH_COOLDOWN_DAYS=16`일
  동안 그 선수에게 재건의 불가 — "주전 건의하자마자 벤치 건의" 같은 모순 방지(사용자 요청).
  UI 버튼 비활성화 + 남은 일수. store `benchCooldown`(playerId→가능일), 시즌말 초기화.

### 2.3 수락 시 — 벤치 지시
- `benchDirectives: { playerId, fromDay }[]` 저장. **부상과 같은 패턴**으로 시즌 계층이
  "그날 출전 후보"에서 제외(로스터엔 남음, 페이롤 유지). 언제든 철회 가능(즉시 복귀).
- 연쇄 효과(이 시스템의 본체):
  - 벤치된 선수 → **경기감각 하락**(FORM_SYSTEM) → 복귀해도 한동안 제 기량이 아님
  - minutes 성향이면 불만 발화 → 면담 대상화(1장)
  - 인기 선수면 팬 분노(3장)

### 3.2 팬심 게이지 (팀 단위, 시즌 누적)
- 시즌 시작 50에서 출발. 오르는 것: 승리·연승·우승 경쟁·스타 활약.
  내리는 것: 연패·**인기 스타의 무단 벤치**(건의로 N경기 연속 제외)·스타 방출.
- 인기 스타 벤치 페널티는 **지속될수록 가속** (3경기 −2, 6경기 −5, 10경기 −10…).
- 뉴스 연동(NewsItem kind `fans`): "팬들, ○○ 연속 결장에 분노 — '구단주는 답하라'",
  "홈 관중 감소세… 싸늘한 객석".

### 3.3 실질 효과 — 돈과 사람으로 돌아온다
- **다음 시즌 예산**: budget × (0.92 + 0.16 × 팬심/100) — 팬심 100이면 +8%, 0이면 −8%(placeholder).
  관중이 줄면 구단주 지갑이 마른다 — 구단주 게임의 체감 손익.
- **선수 이탈 가중**: 팬심 바닥(<25)이면 전 선수 FA 잔류 항에 소폭 −(침몰하는 배 정서).

## 5. 엣지 (미리 박아두는 가드)
- 벤치 지시로 출전 가능 인원이 부족해지면(부상과 겹침) **부상이 우선, 지시는 그 경기 한정 무시**(경기는 성립해야 한다).
- 면담 약속의 이행 판정은 시즌 경계 1회 — 파기 시 다음 시즌 첫 면담 거절 확률에 반영.
- 전원 벤치 지시 같은 악용은 지시 동시 상한(2명)으로 차단.
- 내 팀 전용 기능(AI 구단은 면담·건의 없음 — AI는 이미 수식으로 최적 행동).

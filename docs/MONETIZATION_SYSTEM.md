# 수익화 시스템 (MONETIZATION_SYSTEM) — 광고 · IAP · 엔타이틀먼트

> 무료 출시 + 광고(ad-supported) + 서포터 팩(광고 제거 포함) + DLC(월드컵) 모델의 정본.
> **설계 확정 2026-06-30**(독립 리뷰 1.5단계 거침). 구현은 추후(EAS 개발 빌드 필요 — 네이티브 모듈).
> 핵심 원칙: **광고는 "뜬다(ad-supported)"만 허용, "봐야 넘어간다(ad-gated)"는 금지.** 관전형 1순위·오프라인·안티과금을 깨지 않는다.

## 1. 모델 한 줄

**무료로 받아 풀게임을 하고, 광고가 뜬다. ₩5,000 서포터 팩을 사면 광고가 사라지고 개발을 응원한다. 월드컵은 별도 DLC.**
신생 인디 + 니치 장르라 **도달(무료)** 을 최대화하고, 광고·서포터·DLC가 같은 IAP 인프라를 공유한다.

## 2. 설계 원칙 (이 문서가 지키는 것)

1. **관전형 1순위 불가침** — 광고는 **자동진행·경기 관전의 흐름을 막지 않는다.** 시즌 진행은 광고 응답에 의존하지 않는다.
2. **오프라인 보존** — 네트워크 없어도(지하철·비행기) 플레이·시즌 진행이 멈추지 않는다. 광고는 못 뜨면 그냥 스킵.
3. **안티과금** — 광고제거=편의, DLC=콘텐츠, 서포터=코스메틱. **전부 밸런스 무영향.** 보상형 광고로 전력·스카우팅·자금 이점을 주지 않는다(advantage 금지).
4. **무저장 결정론 격리** — 엔타이틀먼트는 **표시/가용성만** 게이트한다. 시드·리플레이엔 절대 안 들어간다. 엔진은 store 엔타이틀먼트를 직참조하지 않고 명시 파라미터로만 받는다(remove_ads·supporter는 엔진이 안 읽음, worldCup만 WORLDCUP_SYSTEM의 epoch+forward-only로 격리).
5. **유저 관대** — 오프라인 싱글이라 환불 회수가 구조적으로 불가능. 안티치트에 힘 빼지 않는다(false-negative로 정당 구매자 잠그는 게 더 나쁨).

## 3. 광고 (ad-supported)

**배치 = 플레이어가 능동적으로 누르는 "시즌 시작하기" 버튼**(오프시즌 → 새 시즌 킥오프, 2026-06-30 확정).
자동진행·경기 관전 흐름은 **절대 안 건드린다** — 이건 passive 인터럽트가 아니라 player가 의도적으로 새 해를 여는 자연스러운 분기점이라 관전형 1순위와 충돌하지 않는다(독립 리뷰가 반대한 "자동진행 가로채기"와 다름, §9).

- **첫 시즌 무광고** — 첫인상은 깨끗하게(온보딩 마찰 0).
- **이후 "시즌 시작하기" 버튼 탭 시 동영상 광고**(전면/리워드) → 광고 후 새 시즌 시작.
- **오프라인이면 광고 없이 즉시 시작(하드 블록 금지)** — 광고 로드 실패로 시즌이 안 열리면 오프라인 기둥 위반. 못 뜨면 바로 진행. (이때 "비행기모드 광고 회피" 누수가 생기나 싱글플레이라 무시 — 유저 관대, §2.5.)
- **서포터 팩(광고제거) 구매자 = 버튼 눌러도 광고 없이 즉시 시작.**
- **경기 보드·랠리 관전 중 / 자동 시즌진행 중에는 광고 절대 금지**(경기 진행과 무관 — 사용자 명시).
- **(선택) 시간 빈도캡** — 몰아보기(짧은 시즌 연타)가 문제되면 직전 광고 후 N분 캡을 손잡이로 추가. 의도적 버튼이라 필수 아님 — 실측 후 결정.
- **(선택) 정적 메뉴 배너 / 앱오픈 광고** — 멈춘 화면 하단·세션당 1회. 경기보드 금지. 추후 검토.
- 🚫 **금지**: ① 경기 관전·자동진행을 가로채는 광고 ② 보상형 광고로 게임 이점(전력·스카우팅·자금) 해금(안티과금 위반).

## 4. IAP / SKU

전부 **비소모(non-consumable)**. 가격은 잠정.

| SKU | 가격 | 내용 | 안티과금 |
|---|---|---|---|
| `supporter_pack` | ₩5,000 | **광고 제거 + 서포터 뱃지/하트 + 크레딧 등재** (기존 ₩2,000 코스메틱을 이 한 SKU로 통합·승격) | ✅ 편의+코스메틱, 밸런스 0 |
| `dlc_worldcup` | (별도) | 월드컵 4년 국가대표 차출 콘텐츠(WORLDCUP_SYSTEM). 미구매자 비트동일 | ✅ 콘텐츠, 밸런스 0 |

- **결제창 통합**: 광고제거를 별도 SKU로 쪼개지 않고 **서포터 팩 하나에 포함** → 인지부하↓·체감가치↑. "응원하면 광고도 사라지고 개발도 잇는다"(부제 "광고 없이 쾌적하게")로 **따뜻한 응원 프레이밍**. 기존 `supporter.tsx`의 하트·연출 자산 재사용.
- **정직성 조건**: 위 카피는 **광고 게이트를 안 둘 때만** 참이다(게이트 두면 "응원=진행 안 막힘"이라 거짓). §3 비차단이라 성립.

## 5. 엔타이틀먼트 (소유 상태)

- **진실의 원천 = 앱스토어**(StoreKit/Play Billing). 비소모성은 플랫폼이 "구매 복원"·소유권을 무료 보장.
- **로컬 캐시** — 소유 SKU를 기존 save의 **설정군(resetSave에서 살아남는 필드)** 에 둔다(`worldCupOwned`·`supporter` 패턴 1:1). `freshSave`에 넣지 않음 → 기변·세이브 초기화에도 소유 보존. SAVE_SYSTEM version/migrate로 추가.
- **구매 복원 버튼**(필수, 스토어 정책) — 재설치·기변 시 스토어 재조회(온라인 1회). 오프라인 재설치면 일단 미소유로 graceful, **진행 데이터는 절대 삭제 금지**(온라인 되면 복원).
- **엔진 격리** — `removeAds`/`supporter`는 순수 UI 불리언(엔진 미참조). `worldCup`만 sim 영향이나 WORLDCUP_SYSTEM의 epoch+forward-only+비트동일 A/B로 격리.

## 6. 검증 (RevenueCat 하이브리드)

- **RevenueCat** — 영수증 검증·구매 복원·환불 웹훅·iOS/안드 교차. 무료 티어(매출 ~$2.5k/월). 보일러플레이트 절감용.
- **불변 조건**: **출시·시즌 진행은 RevenueCat 응답에 의존하지 않는다.** RC는 구매·갱신 시에만 통신, 진행경로 밖. 오프라인은 로컬 캐시로.
- (리뷰 소견: 비소모성만이라 RC는 다소 오버킬 — 네이티브 복원이 진실의 원천이면 RC 없이도 가능. 무료·편의라 채택 유지. 미래 구독·클라우드 세이브 때 RC 값이 커짐.)
- **Supabase+Vercel 보류** — 수익화엔 불필요. 나중 클라우드 세이브·글로벌 명전(CLAUDE 8)에 투입.

## 7. 빌드 전제

- IAP·AdMob·RevenueCat 전부 **네이티브 모듈 → Expo Go 불가.** **EAS 개발 빌드** 필요(CLAUDE 8 "네이티브 모듈 추가 시 사전 확인").
- SDK 후보: `react-native-google-mobile-ads`(AdMob), `react-native-purchases`(RevenueCat) 또는 `react-native-iap`(네이티브 단독).

## 8. Phase 로드맵

- **P0** EAS 개발 빌드 셋업 + 네이티브 모듈 사전확인(빌드 통과 확인).
- **P1** AdMob — 시즌시작 비차단 전면(첫 시즌 무광고·시간 빈도캡·오프라인 스킵) + (선택)메뉴 배너. **서버 무관 — 단독 검증 가능.**
- **P2** IAP `supporter_pack`(광고 제거 포함) + 로컬 엔타이틀먼트 캐시(save 설정군) + 구매 복원.
- **P3** RevenueCat 연결(검증·복원·환불). 진행경로 불간섭 유지.
- **P4** `dlc_worldcup` — 같은 IAP 인프라 재사용(WORLDCUP_SYSTEM 연동).
- **P5(추후)** Supabase 클라우드 세이브·글로벌 명전(CLAUDE 8).

## 9. 독립 리뷰 기록 (1.5단계, 2026-06-30)

- **기각된 안 — "시즌 진행 광고 하드 게이트"(광고 안 보면 진행 불가)**: 리뷰가 강하게 반대.
  - 이유: ① **관전형 1순위 위반**(가장 손이 안 가야 할 자동진행 지점에 강제 인터럽트, CLAUDE 11) ② **오프라인 치명타**(광고 로드 실패=게임 멈춤) ③ **비행기모드 익스플로잇**(무광고=무료통과 폴백 → 정직한 유저만 광고 보는 벌금) ④ **몰아보기 과세**(20시즌=광고 20번). "advantage 아니니 OK"는 **잘못된 잣대** — 게이트는 밸런스가 아니라 더 윗줄 기둥(관전형·오프라인)을 깬다.
  - → **배치로 해소(§3)**: 광고를 자동진행이 아니라 **플레이어가 능동적으로 누르는 "시즌 시작하기" 버튼**에 붙인다(2026-06-30 사용자 확정). 의도적 분기점이라 관전 흐름·무푸시를 안 깬다. + 오프라인이면 광고 없이 즉시 시작(하드 블록 제거)·경기 진행 무관. 리뷰가 반대한 건 "passive 자동진행 가로채기"였고, 의도적 버튼은 거기 해당 안 됨.
- **채택된 교정**: ① SKU 통합(서포터 팩에 광고제거 포함, ₩5,000) ② 응원 프레이밍 ③ 엔타이틀먼트 표시 전용·결정론 격리 ④ 환불 유저 관대.
- **유지(리뷰가 "써도 무방")**: RevenueCat 하이브리드 — 진행경로 불간섭 조건 하에.

## 10. 미해결 · 추후

- 가격 확정(₩5,000 서포터/DLC), 빈도캡 N분 튜닝(실측), 메뉴 배너·앱오픈 채택 여부.
- Supabase 클라우드(세이브 백업·글로벌 명예의전당·랭킹) — CLAUDE 8, 별도 시점.
- 서버측 영수증 재검증(불법복제 하드닝) — 오프라인 싱글이라 출시 필수 아님, 매출 보이면 검토.

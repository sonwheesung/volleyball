# 스태프 계약 시스템 (STAFF_SYSTEM)

> 단장이 **감독·전문 코치·스카우터**를 스태프 예산 안에서 영입/방출한다. (2026-06-09 신설)
> 결정론 유지: 모든 효과는 순수 함수, 풀은 시드 생성, 계약 상태만 세이브.

## 1. 역할과 효과 — 감독=방향, 코치=상한/수명, 스카우터=정보

| 역할 | 효과 | 통합 지점 |
|---|---|---|
| **감독(head)** | 성향(공/수/밸 → 경기)·카리스마(타임아웃 기세폭)·**훈련선호(어떤 스탯이 크는지=방향)** | `coachInfoOf`·`teamFocus` — 영입 시 시드 감독 대체 |
| **전문 코치(assistant)** | **분야별 다른 효과**(아래). 같은 분야 최고 1명만. **팀당 슬롯 `COACH_SLOTS=3`** | `engine/staff.ts staffEffects` → `evolvePlayer`(훈련·노쇠) |
| **스카우터(scout)** | **내 팀 外 능력 공개도↑**(표시 안개 — 타 구단 선수·드래프트 유망주) + **실제 지명 정확도↑**(공개도 낮으면 가치 오판, AI/단장) | `teamScoutReveal` → `draft.tsx`·선수상세·`RosterList`(표시)·`aiDraftPick`(픽 노이즈) |

**전문 코치 분야별 효과** (현실 코치진 — 더 빨리만이 아니라 더 높이/더 오래):

| 분야 | 효과 | 대상 |
|---|---|---|
| 공격/수비/세터(**기량**) | 해당 스탯 **숨은 포텐셜 상한 +α**(역량100=+5) + 성장 속도↑ | 공격→스파이크·서브 / 수비→리시브·디그·블록 / 세터→세팅·VQ |
| **체력** | **노쇠 하락 둔화**(역량100=−45%, 전성기 연장) + 피지컬 속도↑ | jump·agility·staminaMax·staminaRegen (노쇠 지연은 `DECAY_STATS` 전체) |
| **멘탈** | 집중·기복 **포텐 상한↑** + 멘탈 속도↑ | focus·consistency |

> 포텐 상한 상향(`potBonus`)이 핵심 — 선수를 *진짜 더 잘하게* 만들어 경쟁에 반영(속도 가속만으론 약했음).

> **등급 단조성 실측·봉인(2026-07-07, 검증=Fable 5)**: 코치無 < 역량60 < 역량95 OVR 단조(Δ≈+1.2/3시즌·공격코치 1명), `potRaise(60)=3·potRaise(95)=5`(산식 `round(rating/100×5)` 정확) — `engine/staff.test.ts` "코치 역량 등급 단조성" 테스트(동일 시드 유망주 3갈래 evolvePlayer, 단조 관계 어서션).

## 6. 감독 생애주기 (STAFF_LIFECYCLE, 2026-06-13 설계)

정적 감독 풀(시드 1회 생성·영원)을 **순환하는 생태계**로. `engine/staffLifecycle.ts`(순수·결정론),
오프시즌에 `data/offseason`이 호출. 선수 은퇴(applyRetirements)와 같은 계층.

### 6.1 노쇠·은퇴
- 감독은 매 오프시즌 `age+1`. 선수보다 늦게 은퇴(현장직은 60대까지) — `coachRetireChance(age)`:
  55세↓ 0 · 56~60 0.06 · 61~65 0.18 · 66~70 0.4 · 71+ 0.7 (75+ 0.95). 시드 결정론.
- 은퇴 감독은 풀에서 제거. 소속 감독이 은퇴하면 그 팀은 다음 시즌 새 감독 필요(AI 자동 선임).

### 6.2 신규 공급 — 은퇴 선수 → 지도자
- 선수가 은퇴할 때 **VQ(배구 IQ) 높으면**(≥72) 일정 확률로 **전문 코치(assistant)**로 풀 합류
  (`playerToCoach`). 코치 속성은 선수 커리어에서 파생:
  - **분야(specialty)**: 포지션 매핑 — S→setter · MB→defense · OH/OP→attack · L→defense · (체력/멘탈은 VQ·전성기 길이로 가끔).
  - **역량(rating)**: 선수 VQ + 기량 평균 기반. **카리스마**: VQ + 리더 특성 + 명성(HOF/통산).
  - **style**: 포지션·플레이 성향(공격형 포지션→attack 편향 등).
- **명성 가산**: 레전드(영구결번)·통산 상위 선수는 카리스마·승격 속도 보너스("스타 출신 감독").

### 6.3 승격 — 코치 → 감독
- 전문 코치는 **명성**(retired 선수 시절 + 코치 경력)이 쌓이면 **감독(head) 풀**로 승격(`promotesToHead`):
  - 트리거: ① 코치로 성공(소속 팀 호성적 누적 = coachRep) **또는** ② 본래 명성 높음(레전드 출신).
  - 승격 시 head Coach로 재생성(charisma·style·trainingFocus 계승, 연봉 책정).
- 즉 경로: **은퇴 선수(고VQ) → 전문 코치 → (성과·명성) → 감독**. 레전드는 코치 단계를 빨리 통과.

### 6.4 경질 (성적 부진)
- **시즌 후 경질(AI)**: 최하위(또는 하위 + 장기 부진)면 그 팀 감독 해촉 → 풀로(또는 은퇴) → 새 감독 선임.
  플레이어 팀은 강제 경질 없음(구단주가 결정 — 기존 hireHeadCoach 교체).
- **시즌 중 경질(2026-06-13 요청)**: 시즌 중 성적이 임계 이하로 추락하면 감독 해촉 →
  **전문 코치 중 1명이 감독 대행(acting)**으로 임시 지휘(성향·카리스마는 대행 코치 기준, 약간 저하) →
  다음 영입 시점(또는 시즌 종료)에 정식 감독 선임. 플레이어는 대행 상태에서 직접 감독을 구한다.
  - 구현: 대행은 `headCoachOverride`에 임시 감독(대행 코치를 head로 캐스팅) 주입, `acting:true` 플래그.
- **정식 감독 재직 중 시장 영입 차단(2026-07-11 테스터 정정)**: 예전엔 감독이 있어도 시장에서 새 감독을
  영입하면 **즉시 무경질 교체**됐다(실수로 감독이 바뀌는 사고). → 이제 정식 감독(`head && !acting`)이 있으면
  감독 시장의 "영입" 버튼을 **비활성화 + 안내**("현재 감독을 먼저 경질해야…"), tryHireCoach도 방어적으로
  알림 후 차단. 즉 **교체 = 경질 → 영입 2단계**로 명시화. 대행/공석(`acting || !head`)은 정식 감독을 세우는
  절차라 영입 그대로 허용. 엔진 무관(UI 게이트만) — `hireHeadCoach` 자체는 불변.

### 6.5 결정론·세이브
- 풀 변화(은퇴·신규·승격)는 시드 기반 순수 변환. 세이브엔 계약 상태 + 풀 델타(생성된 코치)만.
- 선수→코치 변환 id는 `staff:{playerId}` 시드로 결정론. 리플레이 일관.

### 6.6 계약·재계약(감독 FA, 2026-06-13 구현)
- `Coach/AssistantCoach.contractYears`: 영입/재계약 시 부여(3년·시드 2~4년), 매 오프시즌 −1.
- 만료(0) 시: **AI**는 성적·연령으로 재계약(`aiResigns`) 또는 FA로 놓아줌 → 다른 팀이 영입(감독 쟁탈).
  **플레이어**는 스태프 화면에서 `재계약(3년)` 또는 새 감독 영입(기존 감독 FA로). 유예 유지(놓치지 않게).
- 경질 인내심: 꼴찌 1년 45% 확률·하위권 2년 연속 확정(`firedEndSeason`+`bottomStreak`) — 매 시즌 경질 과churn 방지.

**구현 현황(2026-06-13)**: ① 엔진 코어 ✅ ② 오프시즌 통합·세이브 영속 ✅ ③ 계약·재계약·FA ✅
④ 장기 검증 ✅(`tools/simStaffLife.ts` — 100시즌 풀 최소 7·연령 안정·감독 79% 선수출신·FA 활성)
⑤ 시즌 중 대행 ✅(`fireCoach` — 경질 시 최고 역량 전문 코치가 대행, 카리스마 ×0.7, 새 감독 영입 시 대행 제거.
   경질 감독은 그 팀 영구 배제·FA로. 스태프 화면: 부진 경고·경질 버튼·대행 표시·영입 유도).
감독 생애주기 전 기능 완성.

## 7. AI 팀 기본 스태프 (2026-06-14 구현)

전엔 AI가 시드 감독뿐이라 코치 부스트·스카우팅 0 → 성장·드래프트 안목에서 일방적으로 불리했다.
이제 **AI 팀마다 결정론 기본 스태프**(전문 코치 2 + 스카우터 1)를 보유한다(`data/league.ts`
`aiTeamAssistants`/`aiTeamScouts`, 팀 id 시드).

- **분리**: AI 스태프는 전용 id(`ai-ac-*`·`ai-sc-*`)라 플레이어 영입 풀과 겹치지 않는다 →
  단일 소속 불변식 유지(플레이어가 AI 코치를 뺏을 수 없고, 그 반대도 없음).
- **레버 보존**: 코치 역량 52~74·공개도 ~43~60%로 *기본기* 수준. 플레이어는 슬롯 3 + 상위 풀
  (역량 최대 92)로 **능가** 가능 — "단장 우위 레버"는 그대로(우위는 거저가 아니라 투자로).
- **효과 적용**: `teamAssistantsOf`/`teamScoutsOf`가 플레이어 팀(=`myTeamStaff`)엔 영입분을,
  나머지엔 AI 기본을 반환 → `evolvePlayer`(성장·노쇠)·`teamScoutReveal`(드래프트)에 그대로 반영.
- **세이브**: AI 스태프는 결정론 파생(미영속) — 구세이브도 로드 시 자동 부여. 플레이어 팀 등록은
  스토어가 `setMyTeamStaff(selectedTeamId)`로 주입(선택·재수화 시).
- **UI 노출(2026-06-14)**: 팀 상세 화면(`app/team/[id].tsx`)에 "코칭 스태프" 섹션 — 전문 코치(분야·역량)·
  스카우터(공개도) 표시. AI 팀은 기본 스태프가, 내 팀은 영입분이 보인다(셀렉터 자동 분기). 읽기 전용
  (영입·방출은 `app/staff.tsx`).

> 밸런스 검증 완료(2026-06-14): KOVO 박스스코어 전 항목 범위 내 불변(N=10,500경기 — day 0 기준이라
> 무영향) · 장기 parity는 오히려 개선(σ 4.06→3.45·왕조 7.0→6.3·우승경험 5.1→5.5/7, 8유니버스×25시즌
> · 엔진 d2f37a8). AI 전 구단 균일 성장으로 랜덤 발산 감소 → 밸런스 회귀 없음. → `sim-league/SKILL.md` 기준선.

### 7.1 플레이어 시작 기본 스태프 (2026-06-24)

AI 팀은 위 기본 스태프가 있는데 **플레이어만 0에서 시작**해 비대칭이었다(스태프 화면 빈 칸 + 초반
성장·스카우팅 불리, 사용자 보고). 이제 `selectTeam` 시 `grantStartingStaff(teamId)`가 플레이어 팀에
**기본 전문코치 1 + 스카우터 1**을 지급한다.
- **결정론 픽**: FA 풀을 역량(코치) / 안목(스카우터)으로 정렬해 **중앙값** 1명씩 자동 영입(예산·슬롯 내,
  멱등 — 이미 있으면 무시). 중위권이라 상위 풀(최대 92) 교체 여지가 남는다 → "단장 우위 레버" 유지.
  실측(team_0): 코치 역량 74·스카우터 78, 스태프 예산 21,900/60,000만(여유 38,100).
- AI 기본 스태프(`ai-*`)와 달리 **FA 풀의 실제 영입분**이라 스태프 화면에 뜨고 **방출·교체 가능**.
  스토어 `staffAssistants`·`staffScouts`에 반영되어 리로드에도 유지. 상세: ONBOARDING_SYSTEM 6.

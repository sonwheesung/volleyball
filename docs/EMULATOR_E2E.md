> 다음에 테스트할 때 **이 문서만 보고 그대로 따라 하면** 같은 검수를 빠르게 재현할 수 있다.

## 0. 공통 사전조건 (모든 케이스 공통 — 한 번 세팅)

| 항목 | 값 / 방법 |
|---|---|
| 전용 AVD | `volleyball` (pixel_6 · android-35). ⚠ **사도전 `emulator-5554`는 절대 건드리지 말 것** |
| 전용 시리얼 | `emulator-5556` — 모든 adb 명령에 `-s emulator-5556` 고정 |
| 개발 서버 | Metro 포트 **8082** (`ANDROID_SERIAL=emulator-5556 npx expo start --android --port 8082`) |
| 앱 로드 | Expo Go 관리형(네이티브 빌드 X). `am start -d exp://127.0.0.1:8082` (+ `adb reverse tcp:8082 tcp:8082`) |
| 화면 보기 | **Bash**(PowerShell `>` 금지 — PNG 깨짐): `"$ADB" -s emulator-5556 exec-out screencap -p > shot.png` → `Read(shot.png)` |
| **좌표 환산(★자주 틀림)** | screencap = 1080×2400. Read가 "×1.20" 안내 → **내가 본 좌표 × 1.20 = 기기 좌표**. `input tap`은 기기 좌표 |
| 탭 | `"$ADB" -s emulator-5556 shell input tap <기기X> <기기Y>` |
| 첫 실행(새 세이브) | 세이브 비우려면 앱 데이터 초기화: `"$ADB" -s emulator-5556 shell pm clear host.exp.exponent` 후 재로드(단, Expo Go도 초기화되니 번들 다시 받음) |

> **진행 규율**(스킬과 동일): 오류(크래시·렌더깨짐·터치무반응·잘린 텍스트·잘못된 수치)는 **도중에 고치고 재시연**.
> 애매(취향·문구 어색)는 **메모만**, 끝에 한꺼번에 질문. 멈춰 묻지 않는다.

## 관찰 메모 (미해결 — 애매·질문 대기)

> 진행 규율상 "틀렸다 단정 불가"한 취향/문구는 고치지 않고 모아 둔다. C1 검수(2026-06-29)에서 나온 2건:

1. **한글 글자 단위 줄바꿈(char-break)** — 스포트라이트 본문이 어절(단어)이 아니라 **글자 단위로 끊긴다**.
   예: `구단들이`→`구단`|`들이`, `감독 몫`, `성적이`→`성`|`적이`, `읽습`|`니다`. 잘림·오버플로는 아님(전문 다 보임).
   → 개선안: 본문 Text에 `word-break: keep-all`(어절 단위 줄바꿈) 적용 검토. **가독성 취향이라 질문 대기.**
2. **캡 초과 빨강 표시** — 시작 로스터가 캡 초과(인천 타이드 37.7억/35.0억=빨강, 잔여 0만). 명문 강팀 시작
   조건상 WAI(새 영입만 막힘, EC-CA-03)이고 빨강 경고는 올바른 동작. **버그 아님 — 기록만.**

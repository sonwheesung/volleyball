# EMULATOR_E2E — 에뮬레이터 실기기 테스트 케이스 (배구명가)

> **이 문서의 용도**: `emulator-test` 스킬이 에뮬레이터로 앱을 띄워 사람 눈으로 검수할 때, **"어느 화면에서 ·
> 무엇을 확인하고 · 어디를 어떤 순서로 탭해야 검증되는지"** 를 케이스 단위로 적어 둔 대본이다.
> 다음에 테스트할 때 **이 문서만 보고 그대로 따라 하면** 같은 검수를 빠르게 재현할 수 있다.
> 스킬 본문(부팅·좌표 환산·진행 규율)은 `.claude/skills/emulator-test/SKILL.md`, 스포트라이트 정본은

| 항목 | 값 / 방법 |

| 전용 시리얼 | `emulator-5556` — 모든 adb 명령에 `-s emulator-5556` 고정 |

| **좌표 환산(★자주 틀림)** | screencap = 1080×2400. Read가 "×1.20" 안내 → **내가 본 좌표 × 1.20 = 기기 좌표**. `input tap`은 기기 좌표 |
| 탭 | `"$ADB" -s emulator-5556 shell input tap <기기X> <기기Y>` |
| 첫 실행(새 세이브) | 세이브 비우려면 앱 데이터 초기화: `"$ADB" -s emulator-5556 shell pm clear host.exp.exponent` 후 재로드(단, Expo Go도 초기화되니 번들 다시 받음) |

## C0 — 첫 경기 전 초기 셋업 무결성 (새 세이브 · 시즌1 · 0경기)

**목적**: 게임을 막 시작한 상태(아직 한 경기도 안 함)에서 **"있으면 안 되는 데이터"가 없는지 · "정상이어야 할 초기값"이
정상인지** 전 화면 + **모든 드릴다운(상세 화면)** 까지 검수. 첫 경기 전인데 통산 기록·시상·부상·이상 폼이 있으면 셋업 버그.

**사전조건**: 새 세이브, 시즌1, currentDay=0(results 비어 있음), 내 구단 확정 직후.

> **검수 규율(이번에 명문화 — 얕은 검수 재발 방지)**: ① **스크롤되는 화면은 끝까지** 스와이프해서 본다(첫 화면만
> 보고 넘기지 않는다). ② **`›`/`TOP 100`/카드/행처럼 누를 수 있는 건 전부 들어가 본다**(목록 화면만 보고 빈 상태라
> 단정 금지 — 상세도 빈지 확인). ③ 빈 상태도 "**빈 상태 메시지가 제대로 뜨는지**"를 본다(흰 화면·깨짐 아님).

> 진행 규율상 "틀렸다 단정 불가"한 취향/문구는 고치지 않고 모아 뒀다가, 사용자 확인 후 일괄 처리했다.

1. **한글 글자 단위 줄바꿈(char-break)** — ~~수정 완료(2026-06-30, `textBreakStrategy='simple'`)~~ →
   **그 fix는 실기기에서 효과 없었음이 2026-07-04 에뮬 검증으로 드러남**(당시 "렌더 전용이라 육안은 다음 사이클" 이라
   **검증을 미뤄 미작동을 몰랐다** — 미검증 fix를 "완료"로 표기한 사각). 실제 원인 2겹: ① Android 라인브레이커(ICU)는
   `textBreakStrategy` 값과 무관하게 **한글 음절 사이를 항상 줄바꿈 가능 지점**으로 봄(‘simple’로 안 바뀜). ② RN 0.81의 `Text`는
   **forwardRef(.render)가 아니라 함수형 컴포넌트**라 컴포넌트 래핑도 불가. → **진짜 fix(2026-07-04, 실기기 검증)**: 웹
   `word-break:keep-all` 격으로 **인접 한글 음절 사이에 WORD JOINER(U+2060) 삽입**. 주입 지점은 **JSX 런타임(jsx/jsxs/jsxDEV)** 을
   전역 1회 감싸 `type===Text`일 때만 children 변형(`lib/koreanLineBreak.ts`·`app/_layout.tsx`). Metro가 `_runtime.jsxDEV`를
   속성 접근으로 호출해 import 순서·구조분해와 무관하게 적용됨. 순수 함수라 **헤드리스 가드 가능**: `tools/_dv_keepall.ts`(A/B 민감도 포함).
   실기기 검증: 스포트라이트 "카드를 누르|면"(글자끊김) → "카드를 / 누르면 선수단과…"(어절 유지). **교훈: 렌더 전용 fix도 반드시
   그 사이클 안에서 육안 검증 — "다음 사이클로 미룸"이 미작동을 8일간 숨겼다.**

# DOCS (문서 담당)

## Role
사도전의 문서를 최신 상태로 유지. `docs/` 가 단일 진실 원천.

## Managed Files
- `CLAUDE.md` / `AGENTS.md`: 프로젝트 종합 명세·룰
- `.claude/.agents/`: 에이전트 역할 문서
- `docs/`: 게임 기획 문서 (00·01·03~17·99)
- `README.md`: 외부 소개

## Rules
- 코드와 문서 간 불일치 즉시 수정 — **게임 메커니즘 변경 시 코드와 docs를 같은 턴에**
- 문서는 간결하고 실용적으로
- 문서 간 상호 링크 사용 (`[15 시작 제자 풀](./15_시작_제자_풀.md)`)
- 신규 문서 추가 시 `docs/README.md` 인덱스 갱신

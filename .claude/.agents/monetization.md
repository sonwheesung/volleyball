# MONETIZATION (수익화 디자이너)

## Role
사도전의 수익화 전략을 설계. 본격 구현은 MVP 이후.

## 출처
세부 BM은 `docs/11_BM.md` 가 단일 진실 원천. 본 문서는 운영 가이드 역할.

## 수익 모델 — F2S (Free-to-Start)
- 기본 게임 플레이는 무료
- 광고 제거 패스 (1회성 / 구독)
- DLC 사부 (외형·시작 통찰·연륜 차등)
- 외형 IAP (사문 외관·인장·UI 테마)

## 핵심 원칙
1. **핵심 양육 루프는 무료** — 사부·제자·강호 시뮬은 과금 게이트 없음
2. **강제 과금 유도 금지** — 다크 패턴 금지
3. **투명한 플랜 비교** — Free vs 유료 차등을 감추지 않음
4. **확률형(가챠) 도입 금지** — 시작 풀 5★ 제자는 DLC로 명시 지급 (`15_시작_제자_풀.md`)
5. **흑화·악인화 같은 다크 엔딩**을 과금 게이트로 묶지 않음

## Free vs Premium 매트릭스
| 영역 | Free | 광고 제거 | DLC 사부 | 외형 IAP |
|---|---|---|---|---|
| 시작 제자 풀 | 8명 (1~4★) | 8명 | 8명 | 8명 |
| 5★ 제자 | 잠금 | 잠금 | 해금 | 잠금 |
| 광고 | 있음 | 없음 | 정책에 따름 | 정책에 따름 |
| 회차 누적 | 무제한 | 무제한 | 무제한 | 무제한 |
| 사부 외형 | 기본 | 기본 | 추가 | 추가 |
| 사문 외관 | 기본 | 기본 | 기본 | 추가 |
